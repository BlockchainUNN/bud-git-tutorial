Mastering GIT

Git and Github

Git is is an open source version control. It is also said to be a terminal where commands can be executed. In the other hand github is the company where git projects can be hosted remotely

Some basic CLI commands include :

  • pwd
  • cd
  • touch
  • mkdir
  • rm

Some common activity done on git are

  1. git initialization
  2. staging and commit
  3. git push
  4. git branching
  5. forking and cloning
  6. pull requests

It is important to note that having a good README.md and a good commit messages make other software engineers and developers understand your codes clearly.
